# Permissions

Simply visit the permissions page for users or user groups (**Settings → Users → User Groups**) to configure these permissions.

## General

### Access AI Assistant

Required for users to access _AI Assistant_ at all.

## AI Assistant

AI Assistant currently does not offer any granular user or user group permissions, but will have some available in the near future.
