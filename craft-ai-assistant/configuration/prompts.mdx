# Prompts

Prompts define the instructions given to the AI when generating content. AI Assistant includes built-in prompts and also allows you to create your own.

<Photo
  img={require('@site/static/img/craft/ai-assistant/v1/configuration/prompts.jpg')}
  alt="AI Prompts"
/>

## Built-in Prompts

AI Assistant includes several ready-to-use prompts for common content generation tasks. You can use these as-is or customize them to fit your needs.

## Custom Prompts

Create your own prompts to control how content is generated.

<VerticalStepWrapper>
  <StepMarkdown>
    Go to **AI Assistant → Prompts** and click **New Prompt**
  </StepMarkdown>
  <StepMarkdown>
    Enter a name and the prompt instructions
  </StepMarkdown>
  <StepMarkdown>
    Optionally bind a prompt to a specific integration.
    :::info
    When a prompt is bound to a specific integration:
    - That integration will be used automatically when the prompt is selected
    - The prompt will only appear when that integration is enabled
    :::
  </StepMarkdown>
  <StepMarkdown>
    Set a sort order value to organize your prompts.
    :::info
    Prompts can be ordered using the sort value to control how they appear in the list. Lower numbers appear first.
    :::
  </StepMarkdown>
  <StepMarkdown>
    Click the **Save** button
  </StepMarkdown>
</VerticalStepWrapper>
