# Getting Started

Use this guide to get productive quickly with AI Assistant after installation and provider setup.

## Before you start

- Install and update the plugin in your Craft project: [Installing & Updating](../setup/installing-updating.mdx)
- Configure at least one provider (API keys + model selection): [Integrations](../configuration/integrations.mdx)
- Review global behavior and field settings: [Settings](../configuration/settings.mdx)
- Optionally customize prompts: [Prompts](../configuration/prompts.mdx)

## Inline AI Button {#inline-ai-button}

The inline **AI** button appears inside supported fields when editing entries, giving you quick access to text and image generation.

- How to use it: [Inline Button](./inline-button.mdx)

## Modal Interface {#modal-interface}

AI Assistant uses a two-column modal experience for generation: input/context on the left and results on the right, with tools to regenerate, insert, or copy.

- Most users access the modal from the inline button inside an entry field: [Inline Button](./inline-button.mdx)
- You can also generate from the dashboard widget: [Dashboard Widget](./widget.mdx)

## Quick AI Actions Widget {#quick-ai-actions-widget}

The **AI Assistant Quick Actions** dashboard widget enables fast, standalone generation without editing an entry.

- Setup and usage: [Dashboard Widget](./widget.mdx)
