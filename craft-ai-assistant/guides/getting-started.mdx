# Getting Started

This guide will help you quickly start generating content with AI Assistant in Craft CMS.

## Using the Inline AI Button

Use the inline AI button directly inside supported fields when editing entries.

<VerticalStepWrapper>
  <StepMarkdown>
    Open an entry with AI-enabled fields. The inline AI button will appear inside the input.
  </StepMarkdown>
  <StepMarkdown>
    Click the button to open the **Generate** modal.
  </StepMarkdown>
  <StepMarkdown>
    Choose a prompt and adjust the **Prompt Text** if needed.
  </StepMarkdown>
  <StepMarkdown>
    (Optional) Enable **Include input context** to use existing field content.
  </StepMarkdown>
  <StepMarkdown>
    Click **Generate** and review the result.
  </StepMarkdown>
  <StepMarkdown>
    When you're satisfied with the result, click **Insert** to replace the field content.
  </StepMarkdown>
</VerticalStepWrapper>

## Quick AI Actions Widget

Use the dashboard widget for quick, standalone generation.

<VerticalStepWrapper>
  <StepMarkdown>
    Add the **AI Assistant Quick Actions** widget to your dashboard.
  </StepMarkdown>
  <StepMarkdown>
    Select a prompt and adjust the **Prompt Text**.
  </StepMarkdown>
  <StepMarkdown>
    Click **Generate**, then copy the result as needed.
  </StepMarkdown>
</VerticalStepWrapper>
