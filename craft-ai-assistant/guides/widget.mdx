# Dashboard Widget

This guide will help you quickly start generating content with AI Assistant using the dashboard widget for quick, standalone generation.

<Photo
  img={require('@site/static/img/craft/ai-assistant/v1/cp/dashboard-widget.jpg')}
  alt="Using the Dashboard Widget"
/>

### Setup Instructions

<VerticalStepWrapper>
  <StepMarkdown>
    Add the **AI Assistant Quick Actions** widget to your dashboard.
  </StepMarkdown>
  <StepMarkdown>
    Select a prompt and adjust the **Prompt Text**.
  </StepMarkdown>
  <StepMarkdown>
    Click **Generate**, then copy the result as needed.
  </StepMarkdown>
</VerticalStepWrapper>
