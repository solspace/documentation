# SugarCRM<Badge type="pro" text="Pro" /><Badge type="feature" text="New in 5.13+" />

This guide assumes you have a [SugarCRM](https://sugarcrm.com) account already.

## Compatibility

Uses `v1.2` of the REST API.

### Endpoints

Maps data to the following endpoints:

- **Contacts**

Does your project require something else? &nbsp; <PrimaryButtonWithLink to='/support/' label='Contact Us' />

### Fields

Maps data to the following field types:

- **Standard**
- **Custom**

## Setup Instructions

<VerticalStepWrapper>
<StepMarkdown>

### Create & get API Key from SugarCRM

- Go to [SugarCRM](https://sugarcrm.com) and log into your account.
- At the top right corner, click on the profile icon and select **Settings**.
- On the next page, on the left side Settings panel, under **SugarCRM API**, click the **API Settings** link.
- You'll then see your keys for SugarCRM API under Account ID and Secret Key. Click the **Generate New API Keys** button if you wish to generate new ones (optional).
- Copy the Account ID and Secret Key.

</StepMarkdown>
<StepMarkdown>

### Set up Integration on your site

- Go to the [Integrations](../configuration/settings.mdx#integrations) section in Freeform Settings (**Freeform** â†’ **Integrations**).
- Find _SugarCRM_ in the list of **service providers**.
- Paste the SugarCRM **Account ID** and **Secret Key** into the **Account ID** and **Secret Key** field, respectively, in Freeform.
- Click the **Save** button.

</StepMarkdown>
<StepMarkdown>

### Authorize the Integration

- After the integration is saved, you will see an **Authorize** button appear.
- Click the **Authorize** button.
- If authorized successfully, you'll see a green _Authorized_ status at the top beside the integration name.

</StepMarkdown>
<StepMarkdown>

### Configure the Form

To use this integration on your form(s), you'll need to configure each form individually.

- Visit the form inside the form builder.
- Click on the **Integrations** tab.
- Click on **SugarCRM** in the list of available integrations.
- On the right side of the page:
  - Enable the integration.
  - Select the API endpoints you wish to map to.
  - Map Freeform fields to the SugarCRM fields as needed.

</StepMarkdown>
</VerticalStepWrapper>
