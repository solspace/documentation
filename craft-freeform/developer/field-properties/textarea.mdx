The `#[TextArea]` property type will render a multi-line text area input field in the field properties panel,
allowing you to define longer text values.

```php showLineNumbers
use Solspace\Freeform\Attributes\Property\Input\TextArea;
// ...

#[TextArea(
    label: 'Lorem Ipsum',
    instructions: 'Enter Lorem Ipsum text here.',
    placeholder: 'Lorem ipsum...',
    rows: 5,
)]
protected ?string $loremIpsum = null;
```
