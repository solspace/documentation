The `#[CodeEditor]` property type will render a code editor in the field properties panel.

```php showLineNumbers
use Solspace\Freeform\Attributes\Property\Input\CodeEditor;
// ...

#[CodeEditor(
    label: 'Twig code',
    instructions: 'Enter your Twig code here.',
    language: 'twig',
)]
protected string $twigCode = '';
```
