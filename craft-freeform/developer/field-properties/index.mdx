# Field Properties<Badge type="feature" text="New in 5.0+" />

When building your own custom field types, integrations or any other module that presents editable
fields in the Freeform form builder, you will find a comprehensive list on available properties and
how to use them here in this document.

Currently, these are the available property types you can add to your fields in the builder:

- [**#[Boolean]**](./boolean) - display a Lightswitch field.
- [**#[Checkboxes]**](./checkboxes) - display a list of checkboxes.
- [**#[CodeEditor]**](./code-editor) - display a code editor with specific syntax highlighting.
- [**#[ColorPicker]**](./color-picker) - display a color picker.
- [**#[DatePicker]**](./date-picker) - display a date picker.
- [**#[DynamicSelect]**](./dynamic-select) - display a dropdown that populates values from an endpoint, capable of also reading other properties to pass to the endpoint
- [**#[DynamicCheckboxes]**](./dynamic-checkboxes) - display a list of checkboxes that populates values from an endpoint, capable of also reading other properties to pass to the endpoint
- [**#[Field]**](./field) - display a dropdown of all the currently available form fields. The fields displayed can be filtered by their implementations.
- [**#[Hidden]**](./hidden) - having a property be available for other fields, but not displaying it in the builder.
- [**#[Integer]**](./integer) - numeric input field.
- [**#[MinMax]**](./min-max) - a two-input combo for setting min and max numeric values.
- [**#[NotificationTemplate]**](./notification-template) - a dropdown letting you select a notification template.
- [**#[Recipients]**](./recipients) - an input that lets the user specify one or more email recipients.
- [**#[Select]**](./select) - a dropdown with predetermined values.
- [**#[TabularData]**](./tabular-data) - lets the user edit tabular data, which is configured by the attribute.
- [**#[Text]**](./text) - an input field.
- [**#[TextArea]**](./textarea) - a textarea field.
- [**#[Wysiwyg]**](./wysiwyg) - a WYSIWYG editor.

For more advanced use-cases, you can also create your own option generators, value generators,
and value transformers. Read more about these in the [Generators and Transformers](./generators-and-transformers) document.
