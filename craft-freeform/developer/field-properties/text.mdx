The `#[Text]` property type will render a simple text input field in the field properties panel,
allowing you to define a single-line text value.

```php showLineNumbers
use Solspace\Freeform\Attributes\Property\Input\Text;
// ...

#[Text(
    label: 'Placeholder Text',
    instructions: 'Define the placeholder text for this field.',
    placeholder: 'Enter placeholder text...',
)]
protected ?string $placeholderText = null;
```
