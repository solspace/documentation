## 4.0.1<TitleDate date="2022-03-18" />

### Fixed
- Fixed a bug where the CP and templates that load Calendar would show PHP8 warning errors.
- Fixed a bug where a calendar's "Allowed Member Groups" settings were not respecting EE6's secondary roles feature.
- Fixed a potential issue with migrating from Calendar 1.x.

## 4.0.0<TitleDate date="2021-05-27" />

### Fixed
- Fixed a few visual errors throughout the Calendar CP.

## 4.0.0-b.3<TitleDate date="2021-05-10" />

### Added
- Added support for the EE6 Menu Manager.

### Changed
- Various visual improvements to the Calendar field type.

### Fixed
- Fixed a potential compatibility issue with other field types.

## 4.0.0-b.2<TitleDate date="2021-05-04" />

### Added
- Added support for the EE6 Jump Menu.

### Fixed
- Fixed various visual errors throughout the Calendar CP and Calendar field type.
- Fixed a bug where Calendar's date picker would conflict with EE's date pickers.
- Fixed a bug where a 'urlSlug' deprecation error would be logged because of Calendar.

## 4.0.0-b.1<TitleDate date="2020-12-24" />

### Added
- Added support for ExpressionEngine 6.
